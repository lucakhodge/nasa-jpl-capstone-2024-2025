Detailed Description

Namespace for the Mars Elevation Model Pathfinding Algorithm (MEMPA).

This namespace includes classes and functions for loading and processing Digital Elevation Model (DEM) data. It provides algorithms to analyze slope to allow an autonomous Mars rover to find efficient paths for navigation, while avoiding risky terrain that could cause accidents.

The MEMPA namespace contains:

  • BUF_DEM For opening and chunking DEM data.
